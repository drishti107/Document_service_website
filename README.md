# Seva Document Centre Website

A responsive informational website for a local document and online application assistance provider.

## Files

- `index.html` — website structure and content
- `styles.css` — responsive design and styling
- `script.js` — navigation, animations, service modal and form handling

## Setup

1. Open `index.html` in a browser to preview the site.
2. Replace all placeholder business information:
   - Business name
   - Phone number
   - WhatsApp number
   - Address
   - Email
3. Search for `919999999999` in `index.html` and replace it with the real WhatsApp number, including country code.
4. Create a Formspree form and replace `YOUR_FORM_ID` in this line:

   `https://formspree.io/f/YOUR_FORM_ID`

5. Upload the three files to GitHub Pages, Netlify, Vercel or any normal web hosting service.

## Privacy note

This version does not collect document uploads. For Aadhaar, PAN, photographs, signatures or certificates, use a secure backend with authentication, encrypted private storage, access controls and a deletion policy. Never collect OTPs, PINs, passwords or banking credentials.
